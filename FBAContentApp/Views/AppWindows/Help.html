Program Overview



The purpose of this program is to create box content labels that meet Amazon fulfillment centers' specifications (you'll need to log into Seller Central to view the documentation).

The program accepts an Excel workbook (.xlsx extenstion) file as the contents of an entire shipment. Each worksheet in the workbook represents a box. The program goes through each worksheet and creates the appropriate box content string. The program creates a Zebra Programming Language (ZPL) string that is sent to the label printer for printing. (NOTE: The printer used must have the capability of interpreting ZPL 2 commands.) All shipment information is saved to a database and labels like the image below are printed.

Box Content Label produced by this program.



The program is loaded with a default Company Ship From address, the one in the image above. In order to reflect your company's information, you must add a company address in the Settings portion of this application. Furthermore, the label also produces the shipment box label that Amazon provides as well. Therefore, you only need this label and not the ones that are generated in Seller Central. However, feel free to use both.

As mentioned previously, when the labels are printed they are also saved to a database. This allows the capability of looking at shipments from the past as well as easily reprinting labels through the History portion of this application. The application also saves shipment contents information in a JSON file to the directory specified in the Settings portion of the application. This is to provide an easy way of doing more with the data if someone chooses to do so.

Creating a Box Content Excel Workbook


This application uses Excel workbooks to read box content information for a shipment. Each worksheet in a workbook represents a box. The box number is determined by the Sheet number. We'll go into more detail on how to prepare this file. If you don't have access to an installation of Excel, you can also prepare this in Google Sheets. As we go through this how-to, examples for both Excel and Google Sheets will be presented.



1. Prepare the file


Using Microsoft Excel

Open a new Excel workbook. Save the workbook. When prompted for the file name, name the file the Shipment Id for the shipment. It usually starts with FBA and a total of 12 characters long. Save it to your desired directory.

Creating new Workbook, saving with the file name the same as Shipment Id. This particular example is saving the file to Desktop.



Using Google Sheets

From the Google Sheets start page, start a new Blank Sheet. In the top left-hand corner, title the sheet as the Shipment Id.
